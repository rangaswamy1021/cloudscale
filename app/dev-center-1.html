Selenium WebDriver
Selenium WebDriver
Selenium is a portable software-testing framework for web applications. It is widely adopted by the QA community for WebUI test automation. It is completely free and works with multiple wide range of browsers.
Where can I get it from?
Selenium HQ
http://www.seleniumhq.org/download/
Maven Repository
https://mvnrepository.com/artifact/org.seleniumhq.selenium
Suggested Tools
Along with Selenium Webdriver you may also need the following:
  • TestNG
  • JUnit
  • Java SDK
  • Eclipse
  • Cucumber
  • Jenkins
